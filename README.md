What is it?
===========

This is a keyboard leds plugin (based on sample plugin) for the Xfce panel.

![Screenshot](xfce4-kbdleds-plugin-screenshot.png)


Installation
============

You should have
- xfce4-dev-tools
- libxfce4panel-2.0-dev
- libxfce4ui-2-dev

Then run
```
./autogen.sh
./configure --prefix=/usr
sudo make install
```

Enjoy!
